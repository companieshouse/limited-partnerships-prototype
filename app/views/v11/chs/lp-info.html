File an update for a limited partnership

Everyone can use this service to view information about limited partnerships.

Authorised Corporate Service Providers (ACSPs) can file updates and confirmation statements for limited partnerships here too. ACSPs are also known as Companies House authorised agents.

This service has replaced paper forms LP6, LP8 and SLP CS01.

How to use the service

You’ll need to:

  • sign in
  • enter the limited partnership's name or number in 'Search the register'
  • select 'File for this company' on the company overview page (you'll only be able to see this if you are signed in and you are an ACSP)
  • select what you'd like to update
