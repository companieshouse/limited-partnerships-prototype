File a limited partnership confirmation statement

Use this service to file a confirmation statement for a limited partnership. This confirms that the limited partnership information we hold on the Companies House register is correct.

! Warning You will not be able to view or change limited partnership information (except for the SIC codes) in this service.

Before filing a confirmation statement, you should check the limited partnership information we hold and update any information that's incorrect.

Updating limited partnership information

To view the information we hold, search for the limited partnership on the Companies House register.

If you need to update any information, select the 'File for this company' button on the limited partnership overview page. You'll need to sign in to your Companies House account.

Who can file for a limited partnership

Only Authorised Corporate Service Providers (ACSPs) can file for a limited partnership. This includes updating information and submitting a confirmation statement. ACSPs are also known as Companies House authorised agents.

Authorised agents are businesses that are registered with a UK Anti-Money Laundering (AML) supervisory body. For example, accountants or solicitors. They must register as an authorised agent with Companies House to be able to file on behalf of clients.
