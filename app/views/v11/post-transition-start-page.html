File for a limited partnership

Use this service to view or update information about a limited partnership, or to file a confirmation statement for a limited partnership.

If you're filing for a limited partnership that was registered before XX DATE, you must provide new required information first.

This service has replaced paper forms LP6, LP8 and SLP CS01.

You must be a Companies House authorised agent to file for a limited partnership

Anyone can view information about limited partnerships, but only Authorised Corporate Service Providers (ACSPs) can file an updates and confirmation statements. ACSPs are also known as Companies House authorised agents.

Authorised agents are businesses like accountants, solicitors or company formation agents, that have registered with Companies House to be able to file on behalf of clients. Find out more about registering as an authorised agent.

If you are someone else involved with the limited partnership, you'll need to find a Companies House authorised agent to file on your behalf.
