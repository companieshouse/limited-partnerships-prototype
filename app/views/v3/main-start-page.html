File for a limited partnership

Limited partnerships must use a Companies House authorised agent to file on their behalf. Authorised agents are also known as Authorised Corporate Service Providers (ACSPs).

Authorised agents can use this service to:

  • update a limited partnership’s details - for example, to change the registered office address
  • update details for a general or limited partner
  • add or remove a general or limited partner
  • file a confirmation statement
  • close the limited partnership

Who can use this service

You, or the business you work for, must be registered as a Companies House authorised agent to use this service.

Read the guidance to find out who can register as an authorised agent.

Finding an authorised agent to file for your limited partnership

Authorised agents are businesses that have registered with Companies House to be able to file on behalf of clients.

To register, they must be supervised for Anti-Money Laundering (AML) purposes. For example, they could be accountants, legal professionals or company formation agents.

When you contact an agent, you should check:

  • if they are registered as an authorised agent with Companies House - this is also known as an Authorised Corporate Service Provider (ACSP)
  • if they can offer the services you need
  • how much they charge for their services

What you'll need

To file for a limited partnership you’ll need:

  • the limited partnership number
  • the authentication code
  • a credit or debit card - if the filing you need to make has a fee

How to get an authentication code

If this is the first online filing for the limited partnership, you’ll need to use this service to file a full update. You must do this by XX DATE.

We'll ask you to provide information that was given when the partnership was registered and new details that are now required by law. Read the guidance on what limited partnership information you need to provide.

When you have submitted the update, we’ll send the authentication code to the registered office address for the limited partnership.

If we have already sent the limited partnership an authentication code, you’ll need to use the service to request a new authentication code.
